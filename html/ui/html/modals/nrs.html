Fork-limited

Blockchain operations are disabled on a fork!

Please, synchronize the blockchain before sending money and/or messages.


Fork should be automatically merged with main blockchain after some time.


If you are still on a fork after a day or two, follow the next steps:
  • Shutdown PrizmCore
  • Erase blockchain:
    • Automatically
      • Start Prizm Configuration Editor
        • Windows: PrizmCore_Launcher.exe
        • MacOS: PrizmEditor.command
        • Linux: Prizm_Launcher.sh
      • Press "I want to Reset Blockchain" checkbox
      • Press "Reset Blockchain" button
      • Confirm by pressing "OK"
    • Manually
      • Locate and remove prizm_db folder:
        on Windows it is found at %USER_HOME%\AppData\Roaming\Prizm
        on Mac and Linux it is usualy found at PrizmCore installation directory
  • Start PrizmCore and wait for blockchain to sync
